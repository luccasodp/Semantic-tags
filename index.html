Semantics

Bold text

To put text in bold, the 'strong' tag is used.

Italic text

To put text in italic, the 'em' tag is used.

Marked text

To put text in marked, the 'mark' tag is used.

Small text

to shorten a text without CSS, the 'small' tag is used.

Deleted text

To put deleted text, use the 'del' tag.

Inserted text

for inserted text, use the 'ins' tag.

Superscript text

for superscript text, use the 'sup' tag.

Subscribed text

for subscribed text, use the 'sub' tag.
